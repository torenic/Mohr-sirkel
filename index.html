Mohrs spenningssirkel

Interaktiv beregning av hovedspenninger, skjærspenning og snittspenninger

Input

Positiv normalspenning = strekk. Negativ normalspenning = trykk. Positiv skjærspenning følger valgt fortegnskonvensjon i diagrammet.

Valgt vinkel: 25°

Mohrs sirkel

Sirkelens skjæring med horisontal akse gir \( \sigma_1 \) og \( \sigma_3 \). Radiusen er \( \tau_{max} \).

Numeriske resultater

Beregningformler

Senter: \[ C = \frac{\sigma_x+\sigma_y}{2} \]
Radius: \[ R = \sqrt{\left(\frac{\sigma_x-\sigma_y}{2}\right)^2+\tau_{xy}^2} \]
Hovedspenninger: \[ \sigma_1 = C+R \] \[ \sigma_3 = C-R \]
Maksimal skjærspenning: \[ \tau_{max}=R \]
Vinkel til hovedplan: \[ \tan(2\theta_p)=\frac{2\tau_{xy}}{\sigma_x-\sigma_y} \]
Spenninger på vilkårlig snitt: \[ \sigma_\theta = C + \frac{\sigma_x-\sigma_y}{2}\cos(2\theta) + \tau_{xy}\sin(2\theta) \] \[ \tau_\theta = -\frac{\sigma_x-\sigma_y}{2}\sin(2\theta) + \tau_{xy}\cos(2\theta) \]
